# Recursos de aprendizaje — canales y términos por tecnología

**Regla importante**: no inventar URLs específicas de videos. Recomendar **canales** + **términos de búsqueda**. El usuario googlea y elige.

## Cómo se usa

En L6, la skill arma `resources.md` del proyecto consultando este archivo, para la tech que el usuario eligió.

## Frontend web

### React
- **Canales YT**: freeCodeCamp, Fireship, Net Ninja, Traversy Media, Jack Herrington, Theo - t3.gg, Web Dev Simplified
- **Términos de búsqueda por fase**:
  - Fundamentos: "react hooks crash course", "useState useEffect explained", "react components props"
  - Práctica: "react custom hooks tutorial", "react performance optimization", "react context vs redux"
  - Proyecto: "react full stack tutorial", "react auth tutorial", "react query tutorial"
- **Libros**: React Docs (react.dev) es la fuente principal — antes era "Road to React" (Robin Wieruch).
- **Repos para Romper**: ejemplo gallery en react.dev, t3-app starter, taxonomy (shadcn).

### Vue

- **Términos**: "vue 3 composition api", "pinia tutorial", "nuxt 3 tutorial"
